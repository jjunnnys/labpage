튜토리얼

이 글은 게시판 기능과 댓글 기능을 알려주는 글입니다.
목차
1. 게시글 작성
2. 댓글 작성
게시글 작성

  1. 글 작성하기

    메인 페이지에서 관리자 확인란 클릭 ex. 현재 시간이 15:17이면 17107호 입력
    ( 맞으면 새로고침이 되면서 다음화면으로 넘어 감, 틀리면 변화없음 )     글쓰기 버튼이 생기면 클릭 글 작성 형식에 맞게 작성 후 작성하기 버튼 누르기
    ( 이전 페이지로 가고 싶으면 뒤로가기 버튼 누르기 )     글 작성이 완료 되면 메인화면으로 이동 됨
    ( 게시물 카드에 글이 등록되 있는 걸 확인 )

  2. 글 수정하기

    수정하고 싶은 글(카드)을 클릭
    게시글 페이지로 들어오면 밑에 수정하기 버튼 클릭
    수정 후 작성하기 버튼 누르면 수정 됨

  3. 글 삭제하기

    수정 버튼을 누른 뒤 글 페이지 나오면 삭제하기 버튼을 누름
댓글 작성

  1. 댓글 작성

    게시글 바닥에 댓글 작성란 확인 댓글 작성을 위한 로그인, 원하는 방식으로 로그인
    ( 오른쪽 위에 있음 )     로그인이 완료되면 사용자 정보가 표시됨 댓글 창에 작성 후 댓글 달기 클릭
    ( 밑에 아이콘들로 원하는 형식으로 작성 )     댓글 확인
    ( '좋아요'와 '싫어요' 가능 )

  2. 댓글 삭제

    본인 댓글의 오른 쪽 끝에 있는 탭을 클릭 후 삭제
    ( 위 사진은 관리자 화면 )
